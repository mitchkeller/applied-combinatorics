In Chapter 6, we discussed Dilworth’s Theorem, which told us that for any poset \(\bfP\) of width \(w\text{,}\) there is a partition of \(\bfP\) into \(w\text{,}\) but no fewer, chains. However, we were only able to devise an algorithm to find this chain partition (and a maximum antichain) in the special case where \(\bfP\) was an interval order. Now, through the magic of network flows, we will be able to devise an efficient algorithm that works in general for all posets. However, to do so, we will require a slightly more complicated network than we devised in the previous section.
