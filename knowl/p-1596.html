Notice that the above theorem does not guarantee that every maximum flow has integer flow on every edge, just that we are able to find one. With this theorem in hand, we now see that if we consider network flow problems in which the capacities are all \(1\) we can find a maximum flow in which every edge carries a flow of either \(0\) or \(1\text{.}\) This can give us a combinatorial interpretation of the flow, in a sense using the full edges as edges that we “take” in some useful sense.
