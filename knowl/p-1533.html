When the augmenting path \(P\) has no backward edges, we set \(\delta= \delta_1\text{.}\) But when \(P\) has one or more backward edges, we pause to set
\begin{equation*} \delta_2 =\min\{\phi(x_{i},x_{i-1}):(x_{i-1},x_i) \text{ a backward edge of } P\}. \end{equation*}
Since every backward edge is used, \(\delta_2>0\) whenever we need to define it. We then set \(\delta=\min\{\delta_1,\delta_2\}\text{.}\)
