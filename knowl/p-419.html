We consider three problems with a common starting point. You are given 1  a set \(S\) of \(10,000\) distinct positive integers, each at most \(100,000\text{,}\) and then asked the following questions.
  1. Is \(83,172\) one of the integers in the set \(S\text{?}\)
  2. Are there three integers in \(S\) whose sum is \(143,297\text{?}\)
  3. Can the set \(S\) be partitioned as \(S=A\cup B\) with \(A\cap B=\emptyset\text{,}\) so that \(\sum_{a\in A}a=\sum_{b\in B}b\text{.}\)
