When \(\PXP\) is an interval order and \(n\) is a positive integer, there may be many different ways to represent \(\bfP\) using intervals with integer end points in \([n]\text{.}\) But there is certainly a least \(n\) for which a representation can be found, and here we see that the representation is unique. The discussion will again make use of the notation for down sets and up sets that we introduced prior to the proof of Dilworth’s Theorem. As a reminder, we repeat it here. For a poset \(\PXP\) and a subset \(S\subset X\text{,}\) let \(D(S) = \{y\in X:\) there exists some \(x\in S\) with \(y\lt x\) in \(P\}\text{.}\) Also, let \(D[S]=D(S)\cup S\text{.}\) When \(|S|=1\text{,}\) say \(S=\{x\}\text{,}\) we write \(D(x)\) and \(D[x]\) rather than \(D(\{x\})\) and \(D[\{x\}]\text{.}\) Dually, for a subset \(S\subseteq X\text{,}\) we define \(U(S) = \{y\in X:\) there exists some \(x\in X\) with \(y>x\) in \(P\}\text{.}\) As before, set \(U[S]=U(S)\cup S\text{.}\) And when \(S=\{x\}\text{,}\) we just write \(U(x)\) for \(\{y\in X:x\lt y\) in \(P\}\text{.}\)
