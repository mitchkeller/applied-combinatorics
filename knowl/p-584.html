We define a recursive algorithm that takes a tree \(\bfT\) on \(k\geq 2\) vertices labeled by elements of a set \(S\) of positive integers of size \(k\) and returns a string of length \(k-2\) whose symbols are elements of \(S\text{.}\) (The set \(S\) will usually be \([k]\text{,}\) but in order to define a recursive procedure, we need to allow that it be an arbitrary set of \(k\) positive integers.) This string is called the Prüfer code of the tree \(\bfT\text{.}\) Let \(\prufer(\bfT)\) denote the Prüfer code of the tree \(\bfT\text{,}\) and if \(v\) is a leaf of \(\bfT\text{,}\) let \(\bfT-v\) denote the tree obtained from \(\bfT\) by removing \(v\) (i.e., the subgraph induced by all the other vertices). We can then define \(\prufer(\bfT)\) recursively by the following procedure.
  1. If \(\bfT\cong \bfK_2\text{,}\) return the empty string.
  2. Else, let \(v\) be the leaf of \(\bfT\) with the smallest label and let \(u\) be its unique neighbor. Let \(i\) be the label of \(u\text{.}\) Return \((i,\prufer(\bfT-v))\text{.}\)
