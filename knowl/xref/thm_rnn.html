Theorem 11.4.

If\(n\) is a positive integer. Then
\begin{equation*} R(n,n) \ge \frac{n}{e\sqrt2} 2^{\frac{1}{2}n} \end{equation*}

Proof.

Let \(t\) be an integer with \(t>n\) and consider the set \(\cgF\) of all labeled graphs with vertex set \(\{1,2,\dots,t\}\text{.}\) Clearly, there are \(2^{C(t,2)}\) graphs in this family. Let \(\cgF_1\) denote the subfamily consisting of those graphs which contain a complete subgraph of sizeÂ \(n\text{.}\) It is easy to see that
\begin{equation*} |\cgF_1|\le \binom{t}{n}2^{n(t-n)}2^{C(t-n,2)}. \end{equation*}
Similarly, let \(\cgF_2\) denote the subfamily consisting of those graphs which contain an independent set of sizeÂ \(n\text{.}\) It follows that
\begin{equation*} |\cgF_2|\le \binom{t}{n}2^{n(t-n)}2^{C(t-n,2)}. \end{equation*}

We want to take the integer \(t\) as large as we can while still guaranteeing that \(|\cgF_1|+|\cgF_2|\le |\cgF|\text{.}\) This will imply that there is a graph \(G\) in \(\cgF\) which does not contain a complete subgraph of size \(n\) or an independent set of sizeÂ \(n\text{.}\) So consider the following inequality:
\begin{equation} 2\binom{t}{n}2^{n(t-n)}2^{C(t-n,2)}\lt 2^{C(t,2)}.\tag{11.4.1} \end{equation}
Now we ask how large can \(t\) be without violating inequalityÂ (11.4.1)? To answer this, we use the trivial inequality \(\binom{t}{n}\le t^n/n!\) and the use the Stirling approximation for \(n!\text{.}\) After some algebra and taking the \(n^{\text{th} }\) root of both sides, we see that we need only guarantee that
\begin{equation*} t\le \frac{n}{e\sqrt{n}}2^{\frac{1}{2}n} \end{equation*}
