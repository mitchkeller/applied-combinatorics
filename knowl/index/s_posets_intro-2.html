A partially ordered set or poset \(\bfP\) is a pair \((X,P)\) where \(X\) is a set and \(P\) is a reflexive, antisymmetric, and transitive binary relation on \(X\text{.}\) (Refer to SectionÂ B.10 for a refresher of what these properties are if you need to.) We call \(X\) the ground set while \(P\) is a partial order on \(X\text{.}\) Elements of the ground set \(X\) are also called points, and the poset \(\bfP\) is finite if its ground set \(X\) is a finite set.
