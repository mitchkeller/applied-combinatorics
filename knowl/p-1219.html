Write each of the following recurrence equations as advancement operator equations.
  1. \(\displaystyle r_{n+2} = r_{n+1}+2r_n\)
  2. \(\displaystyle r_{n+4}=3r_{n+3} - r_{n+2}+2r_n\)
  3. \(\displaystyle g_{n+3} = 5 g_{n+1} - g_n + 3^n\)
  4. \(\displaystyle h_n = h_{n-1} - 2h_{n-2} + h_{n-3}\)
  5. \(\displaystyle r_n = 4r_{n-1} + r_{n-3} - 3 r_{n-5} + (-1)^n\)
  6. \(\displaystyle b_n = b_{n-1} + 3b_{n-2} + 2^{n+1} - n^2\)
