Example 5.40.

Before using Prüfer codes to prove Cayley’s Formula, let’s take a moment to make sure we understand how they are computed given a tree. Consider the \(9\)-vertex tree \(\bfT\) in Figure 5.41.
How do we compute \(\prufer(\bfT)\text{?}\) Since \(\bfT\) has more than two vertices, we use the second step and find that \(v\) is the vertex with label \(2\) and \(u\) is the vertex with label \(6\text{,}\) so \(\prufer(\bfT)=(6,\prufer(\bfT-v))\text{.}\) The graph \(\bfT-v\) is shown in Figure 5.42.
The recursive call \(\prufer(\bfT-v)\) returns \((6,\prufer(\bfT-v-v'))\text{,}\) where \(v'\) is the vertex labeled \(5\text{.}\) Continuing recursively, the next vertex deleted is \(6\text{,}\) which appends a \(4\) to the string. Then \(7\) is deleted, appending \(3\text{.}\) Next \(8\) is deleted, appending \(1\text{.}\) This is followed by the deletion of \(1\text{,}\) appending \(4\text{.}\) Finally \(4\) is deleted, appending \(3\text{,}\) and the final recursive call has the subtree isomorphic to \(\bfK_2\) with vertices labeled \(3\) and \(9\text{,}\) and an empty string is returned. Thus, \(\prufer(\bfT)=6643143\text{.}\)
