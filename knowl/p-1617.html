Let’s see how following this process for the flow in Figure 14.10 leads to a chain partition. If we start with \(x_1'\text{,}\) we see that \((x_1',x_3'')\) is full, so we place \(x_1\) and \(x_3\) in chain \(C_1\text{.}\) Since \(x_3'\) has no flow leaving it, there are no greater elements to add to the chain. However, \(x_1''\) has flow in from \(x_2'\text{,}\) so we add \(x_2\) to \(C_1\text{.}\) We now see that \(x_2''\) has flow in from \(x_7'\text{,}\) so now \(C_1=\{x_1,x_2,x_3,x_7\}\text{.}\) Vertex \(x_7''\) has no flow into it, so the building of the first chain stops. The first vertex we haven’t placed into a chain is \(x_4\text{,}\) so we note that \((x_4',x_5'')\) is full, placing \(x_4\) and \(x_5\) in chain \(C_2\text{.}\) We then look from \(x_5'\) and see no flow leaving. However, there is flow into \(x_4''\) from \(x_6'\text{,}\) so \(x_6\) is added to \(C_2\text{.}\) There is no flow out of \(x_6''\text{,}\) so \(C_2=\{x_4,x_5,x_6\}\text{.}\) Now the first point not in a chain is \(x_8\text{,}\) so we use the flow from \(x_8'\) to \(x_9''\) to place \(x_8\) and \(x_9\) in chain \(C_3\text{.}\) Again, no flow out of \(x_9'\text{,}\) so we look to \(x_8''\text{,}\) which is receiving flow from \(x_{10}''\text{.}\) Adding \(x_{10}\) to \(C_3\) gives \(C_3=\{x_8,x_9,x_{10}\}\text{,}\) and since every point is now in a chain, we may stop.
