A partial order \(P\) is called a total order (also, a linear order) if for all \(x,y\in X\text{,}\) either \(x\le y\) in \(P\) or \(y\le x\) in \(P\text{.}\) For small finite sets, we can specify a linear order by listing the elements from least to greatest. For example, \(L=[b,c,d,a,f,g,e]\) is the linear order on the ground set \(\{a,b,c,d,e,f,g\}\) with \(b\lt c\lt d\lt a\lt f\lt g\lt e\) in \(L\text{.}\)
