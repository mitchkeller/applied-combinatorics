We’ve seen several expressions that can be used to calculate the binomial coefficients, but in order to extend \(C(p,k)\) to real values of \(p\text{,}\) we will utilize the form
\begin{equation*} \binom{p}{k}=\frac{P(p,k)}{k!}, \end{equation*}
recalling that we’ve defined \(P(p,k)\) recursively as \(P(p,0)=1\) for all integers \(p\geq 0\) and \(P(p,k)=p P(p-1,k-1)\) when \(p\geq k > 0\) (\(k\) an integer). Notice here, however, that the expression for \(P(p,k)\) makes sense for any real number \(p\text{,}\) so long as \(k\) is a non-negative integer. We make this definition formal.
