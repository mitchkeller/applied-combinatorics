Many authors prefer to start the development of number systems with the set of positive integers and defer the introduction of the concept of zero. In this setting, you have a non-empty set \(\posints\text{,}\) a one-to-one successor function \(s:\posints\injection\posints\) and a positive integer called one and denotedÂ \(1\) that is not the successor of any positive integer. The Principle of Induction then becomes: If \(\mathbb{M}\subseteq\posints\text{,}\) then \(\mathbb{M} =\posints\) if and only if
  1. \(1\in \mathbb{M}\text{;}\) and
  2. \(\forall k\in \nonnegints\quad(k\in \mathbb{M}) \Longrightarrow(s(k)\in \mathbb{M})\text{.}\)
