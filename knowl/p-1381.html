We recall that the binomial coefficients satisfy
\begin{equation*} \binom{m+n-2}{m-1}=\binom{m+n-3}{m-2}+\binom{m+n-3}{m-1} = \binom{m+n-3}{m-2}+\binom{m+n-3}{n-2} \end{equation*}
So either \(|S_1|\ge \binom{m+n-3}{m-2}\) or \(|S_1|\ge\binom{m+n-3}{m-1}\text{.}\) If the first option holds, and \(S_1\) does not have an independent set of size \(n\text{,}\) then it contains a complete subgraph of size \(m-1\text{.}\) It follows that we may add \(x\) to this set to obtain a complete subgraph of size \(m\) in \(G\text{.}\)
