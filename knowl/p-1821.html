Let \(\cgF\) be a finite family of events in a probability space and for each event \(E\in\cgF\text{,}\) let \(\cgN(E)\) denote a subfamily of events from \(\cgF-\{E\}\) so that \(E\) is independent of any event not in \(\cgN(E)\text{.}\) Suppose that for each event \(E\in\cgF\text{,}\) there is a real number \(x(E)\) with \(0\lt x(E)\lt 1\) such that
\begin{equation*} P(E)\le x(E)\prod_{F\in\cgN(E)}(1-x(F)). \end{equation*}
Then for every non-empty subfamily \(\cgG\subseteq\cgF\text{,}\)
\begin{equation*} P(\prod_{E\in\cgG}\overline{E})\ge\prod_{E\in\cgG}(1-x(E)). \end{equation*}
In particular, the probability that all events in \(\cgF\) fail is positive.
