Given an indexed family of sets \(\cgF=\{S_\alpha:\alpha\in V\}\text{,}\) we associate with \(\cgF\) a graph \(\bfG\) defined as follows. The vertex set of \(\bfG\) is the set \(V\) and vertices \(x\) and \(y\) in \(V\) are adjacent in \(\bfG\) if and only if \(S_x\cap S_y \neq\emptyset\text{.}\) We call \(\bfG\) an intersection graph. It is easy to see that every graph is an intersection graph (Why?), so it makes sense to restrict the sets which belong to \(\cgF\text{.}\) For example, we call \(\bfG\) an interval graph if it is the intersection graph of a family of closed intervals of the real line \(\reals\text{.}\) For example, in FigureÂ 5.27, we show a collection of six intervals of the real line on the left. On the right, we show the corresponding interval graph having an edge between vertices \(x\) and \(y\) if and only if intervals \(x\) and \(y\) overlap.
