When \(\PXP\) is a poset and \(C\) is a subset of \(X\text{,}\) we say that \(C\) is a chain if every distinct pair of points from \(C\) is comparable in \(P\text{.}\) When \(P\) is a linear order, the entire ground set \(X\) is a chain. Dually, if \(A\) is a subset of \(X\text{,}\) we say that \(A\) is an antichain if every distinct pair of points from \(A\) is incomparable in \(P\text{.}\) Note that a one-element subset is both a chain and an antichain. Also, we consider the empty set as both a chain and an antichain.
