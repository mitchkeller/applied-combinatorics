The number of integer solutions is:
  1. \(C(537,5)\text{,}\) when all \(x_i> 0\) and equality holds;
  2. \(C(543,5)\text{,}\) when all \(x_i\ge 0\) and equality holds;
  3. \(C(291,3)\text{,}\) when \(x_1,x_2,x_4,x_6>0\text{,}\) \(x_3=52\text{,}\) \(x_5=194\text{,}\) and equality holds;
  4. \(C(537,6)\text{,}\) when all \(x_i > 0\) and the inequality is strict (Imagine a new variable \(x_7\) which is the balance. Note that \(x_7\) must be positive.);
  5. \(C(543,6)\text{,}\) when all \(x_i \ge 0\) and the inequality is strict (Add a new variable \(x_7\) as above. Now it is the only one which is required to be positive.); and
  6. \(C(544,6)\text{,}\) when all \(x_i \ge 0\text{.}\)
