A flow \(\phi\) in a network is a function which assigns to each directed edge \(e=(x,y)\) a non-negative value \(\phi(e)=\phi(x,y)\leq c(x,y)\) so that the following conservation laws hold:
  1. \(\sum_{x} \phi(S,x)= \sum_{x} \phi(x,T)\text{,}\) i.e., the amount leaving the source is equal to the amount arriving at the sink. This quantity is called the value of the flow \(\phi\text{.}\)
  2. For every vertex \(y\) which is neither the source nor the sink the amount leaving \(y\) is equal to the amount entering \(y\text{.}\) That is, \(\sum_{x}\phi(x,y)= \sum_{x}\phi(y,x)\text{.}\)
