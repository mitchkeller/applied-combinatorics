Note that we can consider \(P\) to be extended to a mapping from \(S\) to \([0,1]\) by setting \(P(x)=P(\{x\})\) for each element \(x\in S\text{.}\) We call the elements of \(S\) outcomes (some people prefer to say the elements are elementary outcomes) and the quantity \(P(x)\) is called the probability of \(x\text{.}\) It is important to realize that if you know \(P(x)\) for each \(x\in S\text{,}\) then you can calculate \(P(E)\) for any event \(E\text{,}\) since (by the second property), \(P(E)=\sum_{x\in X}P(x)\text{.}\)
