In Alice’s game as described above, the set \(S\) can be \(\{0,1,2,3,4,5\}\text{,}\) the set of possible differences when a pair of dice are rolled. In this game, we will see that the correct definition of the function \(P\) will set \(P(0)=6/36\text{;}\) \(P(1)=10/36\text{;}\) \(P(2)=8/36\text{;}\) \(P(3)=6/36\text{;}\) \(P(4)=4/36\text{;}\) and \(P(5)=2/36\text{.}\) Using Xing’s more compact notation, we could say that \(P(0)=1/6\) and \(P(d)= 2(6-d)/36\) when \(d>0\text{.}\)
