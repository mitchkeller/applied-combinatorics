Example 5.43.

We close this section with an example of how to take a Prüfer code and use it to construct a labeled tree. Consider the string \(\bfs=75531\) as a Prüfer code. Then the tree \(\bfT\) corresponding to \(\bfs\) has \(7\) vertices, and its leaves are labeled \(2\text{,}\) \(4\text{,}\) and \(6\text{.}\) The inductive step in our proof attaches the vertex labeled \(2\) to the vertex labeled \(7\) in the tree \(\bfT'\) with Prüfer code \(5531\) and vertex labels \(\{1,3,4,5,6,7\}\text{,}\) since \(2\) is used to label the last vertex added. What are the leaves of \(\bfT'\text{?}\) The symbols in \(\{4,6,7\}\) do not appear in \(5531\text{,}\) so they must be the labels of leaves, and the construction says that we would attach the vertex labeled \(4\) to the vertex labeled \(5\) in the tree we get by induction. In Figure 5.44, we show how this recursive process continues.
We form each row from the row above it by removing the first label used on the edge added from the label set and removing the first symbol from the Prüfer code. Once the Prüfer code becomes the empty string, we know that the two remaining labels must be the labels we place on the ends of \(\bfK_2\) to start building \(\bfT\text{.}\) We then work back up the edge added column, adding a new vertex and the edge indicated.
